Instructions:

1)Run GUI.py file(it is the main file).
2)Now, the main GUI window will open with the news content.
3)You can try to use the different options on the GUI window.

NOTE:
1)No installation needed.
2)If you click the refresh button on the GUI window,
  it will work in the backend for pulling the data from 
  news websites(this will suspend the window for a while).
  After completion of that process you can view the new data
  on the GUI window.
